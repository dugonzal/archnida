## Installation

### Linux

```bash
git clone git@github.com:dugonzal/archnida.git

cd archnida

python3 -m venv venv

source venv/bin/activate or . venv/bin/activate

pip install -r requirements.txt

python archnida.py
```

### Windows

```bash

git clone git@github.com:dugonzal/archnida.git

cd archnida

python -m venv venv or py -m venv venv

venv\Scripts\activate.bat

pip install -r requirements.txt

python archnida.py
```
